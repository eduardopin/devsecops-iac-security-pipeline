# devsecops-iac-security-pipeline

> Public portfolio project by Eduardo Pin.  
> Sanitized, non-company, no secrets, no proprietary code.

## Purpose

DevSecOps IaC pipeline using Terraform, Checkov, Trivy, Semgrep, OPA and SARIF reporting

## What this repository demonstrates

- Cloud architecture and platform engineering decision-making
- Infrastructure as Code and automation patterns
- Reliability, security, cost governance and operational maturity
- Clear documentation for technical recruiters and engineering leaders

## Repository structure

```
.
├── docs/                  # architecture, ADRs, diagrams, operating model
├── terraform/             # IaC examples and modules
├── modules/               # reusable building blocks
├── scripts/               # automation scripts
├── runbooks/              # operational procedures and incident response
├── dashboards/            # observability examples
├── src/                   # application or automation code when applicable
└── .github/workflows/     # CI/CD, validation and security gates
```

## Status

Initial portfolio structure. This repo is intended to evolve with concrete examples, diagrams, runbooks and automation.

## Security note

This repository does not contain production credentials, customer data, company-specific architecture, proprietary code or private infrastructure details.
